# eslint-config

## Installation

Install with `yarn` or `npm`:

```bash
yarn add -D @ajliv/eslint-config
```

Then, add to the "extends" array in your .eslintrc.\* file. Make sure to put it last, so it gets the chance to override other configs.

```json
{
  "extends": ["@ajliv"]
}
```
